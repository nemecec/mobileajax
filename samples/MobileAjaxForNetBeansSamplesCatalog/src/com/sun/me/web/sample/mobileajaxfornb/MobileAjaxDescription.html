Mobile Ajax contains five sample projects that are hosted at https://meapplicationdevelopers.dev.java.net/mobileajax.html.
  • Java ME GPS Application
  • Java ME client for Flickr
  • Java ME client for local business search
  • Blog reader and publisher for Java ME
  • Java ME client for twitter.com
There are also four libraries projects atom, json, path, request that are necessary for these sample projects.
Every project contains README file that describes its functionality. The README will open in editor.
IMPORTANT NOTE: some of the services used in the samples needs an ID. You have to register to them before the sample will work correctly. Read the README carufully, please.
